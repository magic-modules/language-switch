@magic-modules/language-switch

dies ist das @magic-modules LanguageSwitch modul. Es rendert ein Menu mit Sprachauswahl.

installation:

npm install @magic-modules/language-switch

usage:

in einer page oder einem komponent

export const View = state => div({ class: 'page' }, LanguageSwitch(state))

der source dieser page ist im example directory und wird mit @magic/core gebaut und zu github publiziert.
